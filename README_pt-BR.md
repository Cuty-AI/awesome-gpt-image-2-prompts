## 🤖 Conheça o GPT Image 2

O **GPT Image 2** é o modelo de imagem multimodal nativo de próxima geração da OpenAI — sucessor do GPT-Image-1. Ele trata a *linguagem* como a principal superfície de controle e brilha justamente onde os modelos anteriores tropeçavam.

O que ele faz excepcionalmente bem:

| Capacidade | Por que importa |
|---|---|
| 🔤 **Renderização literal de texto** | Pôsteres, infográficos, embalagens, mockups de UI — as palavras saem *corretas*, não embaralhadas. |
| 🧠 **Seguir instruções longas** | Briefs de vários parágrafos com restrições, layouts e paletas de cor são respeitados. |
| 🪄 **Edição ancorada em referência** | Envie uma ou mais imagens e peça mudanças cirúrgicas — pose, figurino, iluminação, props. |
| 🧑‍🤝‍🧑 **Consistência de personagem** | Mesmo personagem, mesmo estilo, atravessando quadros, cenas e storyboards. |
| 📐 **Precisão de composição** | Grades bento, colagens 2×2, mapas isométricos, vistas explodidas, layouts no estilo knolling. |
| 🎚️ **Fidelidade de estilo** | Aquarela, risografia, low-poly, blueprint, próximo do Studio Ghibli — sem aquele "verniz de IA". |
