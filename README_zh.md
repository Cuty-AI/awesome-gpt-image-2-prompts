## 🤖 认识 GPT Image 2

**GPT Image 2** 是 OpenAI 新一代原生多模态图像模型 —— GPT-Image-1 的继任者。它把 *自然语言* 作为主要的控制接口，在以往模型频频失手的场景中大放异彩。

它最擅长的能力：

| 能力 | 为什么重要 |
|---|---|
| 🔤 **精准文字渲染** | 海报、信息图、包装、UI 原型 —— 文字写得 *准确*，不会糊成乱码。 |
| 🧠 **长指令理解** | 多段落的复杂需求 —— 约束、版式、调色板 —— 都能被完整尊重。 |
| 🪄 **基于参考图的编辑** | 上传一张或多张图，做精细化修改 —— 姿势、服装、光线、道具皆可。 |
| 🧑‍🤝‍🧑 **角色一致性** | 同一个角色、同一种风格，在分镜、场景、故事板之间稳定保持。 |
| 📐 **构图精度** | 便当格、2×2 拼图、等距地图、爆炸视图、整齐摆放（knolling）布局。 |
| 🎚️ **风格还原度** | 水彩、Risograph、低多边形、蓝图、吉卜力风 —— 不带"AI 油腻感"。 |
