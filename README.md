## 🤖 Meet GPT Image 2

**GPT Image 2** is OpenAI's next-generation native multimodal image model — the successor to GPT-Image-1. It treats *language* as the primary control surface and shines wherever past models stumbled.

What it does exceptionally well:

| Capability | Why it matters |
|---|---|
| 🔤 **Verbatim text rendering** | Posters, infographics, packaging, UI mockups — words come out *right*, not garbled. |
| 🧠 **Long-instruction following** | Multi-paragraph briefs with constraints, layouts, and color palettes are respected. |
| 🪄 **Reference-grounded editing** | Upload one or more images and ask for surgical changes — pose, outfit, lighting, props. |
| 🧑‍🤝‍🧑 **Subject consistency** | Same character, same style, across panels, scenes, and storyboards. |
| 📐 **Compositional precision** | Bento grids, 2×2 collages, isometric maps, exploded views, knolling layouts. |
| 🎚️ **Style fidelity** | Watercolor, risograph, low-poly, blueprint, Studio-Ghibli-adjacent — without "AI sheen." |
