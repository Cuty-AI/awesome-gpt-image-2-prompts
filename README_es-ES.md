## 🤖 Conoce GPT Image 2

**GPT Image 2** es el modelo de imágenes multimodal nativo de próxima generación de OpenAI — el sucesor de GPT-Image-1. Trata el *lenguaje* como la superficie principal de control y brilla justo donde los modelos anteriores tropezaban.

En lo que destaca de verdad:

| Capacidad | Por qué importa |
|---|---|
| 🔤 **Renderizado de texto literal** | Pósters, infografías, packaging, mockups de UI — las palabras salen *bien*, no como garabatos. |
| 🧠 **Seguimiento de instrucciones largas** | Briefs de varios párrafos con restricciones, layouts y paletas de color se respetan al pie de la letra. |
| 🪄 **Edición anclada en referencias** | Sube una o varias imágenes y pide cambios quirúrgicos — pose, vestuario, iluminación, props. |
| 🧑‍🤝‍🧑 **Consistencia de personaje** | El mismo personaje, el mismo estilo, a través de viñetas, escenas y storyboards. |
| 📐 **Precisión compositiva** | Cuadrículas bento, collages 2×2, mapas isométricos, vistas explotadas, layouts tipo knolling. |
| 🎚️ **Fidelidad de estilo** | Acuarela, risografía, low-poly, blueprint, cercano al estilo Studio Ghibli — sin el típico "brillo a IA". |
