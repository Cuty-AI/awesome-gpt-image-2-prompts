## 🤖 GPT Image 2 소개

**GPT Image 2** 는 OpenAI 의 차세대 네이티브 멀티모달 이미지 모델로, GPT-Image-1 의 후속 모델입니다. *언어* 를 기본 제어 인터페이스로 다루며, 이전 모델들이 자주 무너졌던 영역에서 빛을 발합니다.

특히 잘하는 일:

| 역량 | 왜 중요한가 |
|---|---|
| 🔤 **글자 그대로의 텍스트 렌더링** | 포스터·인포그래픽·패키지·UI 목업에서 글자가 *정확하게* 나옵니다. 깨지지 않습니다. |
| 🧠 **긴 지시문 준수** | 여러 단락에 걸친 브리프 — 제약, 레이아웃, 컬러 팔레트까지 그대로 따라줍니다. |
| 🪄 **레퍼런스 기반 편집** | 한 장 또는 여러 장의 이미지를 올리고 포즈·의상·조명·소품에 외과적 수정을 요청할 수 있습니다. |
| 🧑‍🤝‍🧑 **캐릭터 일관성** | 같은 캐릭터, 같은 스타일을 컷·신·스토리보드 전반에 걸쳐 유지합니다. |
| 📐 **구도의 정밀함** | 벤또(Bento) 그리드, 2×2 콜라주, 아이소메트릭 맵, 분해도, 놀링(knolling) 레이아웃까지. |
| 🎚️ **스타일 충실도** | 수채화, 리소그래프, 로우폴리, 블루프린트, 지브리풍 — 흔한 "AI 광택" 없이. |
