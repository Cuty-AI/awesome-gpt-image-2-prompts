## 🤖 Hallo, GPT Image 2

**GPT Image 2** ist OpenAIs natives multimodales Bildmodell der nächsten Generation — der Nachfolger von GPT-Image-1. Es behandelt *Sprache* als primäre Steuerebene und glänzt überall dort, wo frühere Modelle stolperten.

Wo es besonders überzeugt:

| Fähigkeit | Warum es zählt |
|---|---|
| 🔤 **Wortgetreues Textrendering** | Poster, Infografiken, Verpackung, UI-Mockups — die Wörter kommen *richtig* heraus, nicht als Buchstabensalat. |
| 🧠 **Lange Instruktionen verstehen** | Mehrseitige Briefings mit Einschränkungen, Layouts und Farbpaletten werden respektiert. |
| 🪄 **Referenzbasierte Bearbeitung** | Lade ein oder mehrere Bilder hoch und fordere chirurgisch genaue Änderungen an — Pose, Outfit, Licht, Props. |
| 🧑‍🤝‍🧑 **Charakter-Konsistenz** | Derselbe Charakter, derselbe Stil — über Panels, Szenen und Storyboards hinweg. |
| 📐 **Kompositorische Präzision** | Bento-Raster, 2×2-Collagen, isometrische Karten, Explosionszeichnungen, Knolling-Layouts. |
| 🎚️ **Stiltreue** | Aquarell, Risograph, Low-Poly, Blueprint, Studio-Ghibli-nah — ohne "KI-Lack". |
