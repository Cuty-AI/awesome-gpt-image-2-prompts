## 🤖 GPT Image 2 とは

**GPT Image 2** は、OpenAI のネイティブなマルチモーダル画像モデルの次世代版であり、GPT-Image-1 の後継です。*言語* を主要な操作インターフェースとして扱い、これまでのモデルがつまずいてきた領域で真価を発揮します。

特に強いポイント：

| 能力 | なぜ重要か |
|---|---|
| 🔤 **文字を文字通り描く** | ポスター、インフォグラフィック、パッケージ、UI モック ―― 文字が *正しく* 出る。崩れない。 |
| 🧠 **長文プロンプトへの追従** | 複数段落にまたがる指示、制約、レイアウト、カラーパレットまでしっかり尊重される。 |
| 🪄 **参照画像ベースの編集** | 1 枚または複数の画像をアップして、ポーズ・衣装・ライティング・小物に外科的な変更が可能。 |
| 🧑‍🤝‍🧑 **キャラクターの一貫性** | 同じキャラを同じタッチで、コマ・シーン・絵コンテをまたいで保てる。 |
| 📐 **構図の精密さ** | 弁当グリッド、2×2 コラージュ、アイソメトリックマップ、分解図、ノーリング・レイアウトなど。 |
| 🎚️ **スタイル再現度** | 水彩、リソグラフ、ローポリ、ブループリント、ジブリ風 ―― 例の「AI っぽい光沢」抜きで。 |
